# Email Verifier V2 Web App

This React + Vite frontend is the operator UI for Email Verifier V2.

## Features

- submit direct-callout verifications
- inspect classification, confidence, and protocol evidence
- review enrichment evidence for `accept_all` and `unknown`
- browse verification history and stats
- manage API keys and admin views

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```
